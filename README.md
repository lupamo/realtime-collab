# REAL-TIME COLLABORATIVE TASK MANAGEMENT PLATFORM
A fullstack application showcasing microservices architecture, real-time features, and modern developmennt practices.

## Tech Stack
### Backend
- Python 3.11 with FastAPI
- PostgreSQL for primary data storage
- Redis for caching and pub/sub messaging
- Celery for background task processing
- SQLAlchemy ORM with ALembic migrations
- JWT authentication with refresh tokens

### Frontend
- Nextjs (App Router)
- TypeScript for type safety
- Tailwind CSS + shadcn/ui
- Tanstack Query for state management
- WebSocket client for real-time updates

### Infrastructure
- Docker & Docker Compose
- NGINX as api Gateway
- Multi-service architecture with health checks
